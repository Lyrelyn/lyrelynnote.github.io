## Why a blog?

I've been meaning to start a blog for a while. Writing helps me organize my thoughts, solidify what I've learned, and share knowledge with the community. This space is where I'll document my journey in software development, share technical deep-dives, and showcase projects.

## What to expect

I'll be writing about:

- **Web development** — React, Next.js, TypeScript, and modern frontend
- **Open source** — Projects I'm building and contributing to
- **Learning in public** — Notes, tutorials, and things I figure out along the way
- **Project walkthroughs** — Breaking down how things are built

### Tech stack of this blog

This blog itself is built with:

- **Next.js 15** (App Router) — for server-side rendering and static generation
- **TypeScript** — because types make everything better
- **Tailwind CSS 4** — for utility-first, responsive design
- **MDX** — Markdown with embedded React components
- **Vercel** — for seamless deployment

<Callout type="info">
  This blog is a work in progress! I'll be adding features like search, more custom MDX components, and polish over time.
</Callout>

## Let's go

Whether you're here to learn something new, get inspired, or just browse — I'm glad you stopped by. Stay tuned for more content coming soon.
