This very blog you're reading! Built from scratch with modern web technologies.

## Features

- **Next.js 15** with App Router and Server Components
- **TypeScript** for type safety throughout
- **Tailwind CSS 4** with custom design tokens
- **MDX** content with custom components (Callout, CodeBlock)
- **Dark mode** with system preference detection
- **RSS feed** and **sitemap** for SEO
- **Giscus** comments powered by GitHub Discussions
- **Framer Motion** for smooth page transitions
- Automatic deployment on **Vercel**

## Architecture

The blog uses a filesystem-based content approach. All posts are `.mdx` files in the `content/blog/` directory, with frontmatter for metadata. Posts are loaded at build time for optimal performance.
